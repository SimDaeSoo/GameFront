Game Project / Front End
=============
Game Front Server
멀티 슈팅게임 ADI의 Front 서버로서, Client에 해당한다.

Example URL : http://13.124.180.130:7000/

### Controll ###
Arrow Key : Moving  
'M' : Zoom In / Zoom Out  
'Enter' : Chat (TODO)  

### How To Run? ###
- dependency install
```sh
$ npm install
```
- build and run as develop mode
```sh
$ npm start
```
- build and run as production mode
```sh
$ npm run build
```

### Server ###
- AWS EC2 Instance
- PM2
- Node.js
- Express
- Socket.io

### Front Dev Environments ###
- Vue.js
- PIXI.js
- Typescript
- Javascript
- Webpack 4
